# Hedge-Fund-Strategy-Dashboard-ML-Prototype

📌 Overview

This project was developed as part of Karyaneeti – National Case Competition (NSUT E-Summit 2025), where the solution secured 13th rank among 4,000+ participants.

The objective of the case study was to analyze hedge fund performance over time and identify the most suitable hedge fund(s) for investment using data analytics and machine learning.

🎯 Problem Statement

Given historical hedge fund performance data, the task was to:

Compare multiple hedge funds across time
Analyze risk and return characteristics
Support investment decision-making using quantitative metrics
🛠️ Solution Approach

The solution follows an end-to-end analytics pipeline:

Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature engineering for performance and risk metrics
Dashboard-based comparative analysis
Machine learning prototype for fund evaluation
📊 Dashboard Insights

An interactive dashboard was built to compare hedge funds based on:

Returns and cumulative performance
Volatility and drawdowns
Risk-adjusted metrics (Sharpe-like indicators)
Time-based fund performance comparison
Dashboard enables identifying hedge funds that are consistently suitable for investment over time.

🤖 Machine Learning Prototype

A machine learning model was developed to:

Analyze historical performance patterns
Support data-driven fund selection
Assist in evaluating hedge fund strategies
Note: This ML prototype is intended as a decision-support tool, not a fully automated trading system.
🧰 Tech Stack

Python (Pandas, NumPy, Matplotlib)
Machine Learning (scikit-learn)
Power BI (Dashboarding & Visualization)

🚀 Results & Outcomes

Enabled structured comparison of hedge funds across time
Supported investment suitability analysis using quantitative metrics
Delivered a data-driven solution for a real-world hedge fund case study
